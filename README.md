
## Simple app to manage notes in markdown format ##

### install app: ###

Create app.json config file in /config based on example file.

```
$ cd client && npm install && bower install
```

### run app: ###

```
$ nodemon app.js
$ cd client && grunt dev
```

Open: **http://localhost:9000**

### deploy: ###
* `/client/app`
* `/server`
* `/config`
* `app.js`
